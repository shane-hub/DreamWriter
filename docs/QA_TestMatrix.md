# QA Test Matrix (左移切入测试矩阵)

## 1. 测试范围 
本测试矩阵旨在验证 **DreamWriter** 前端与 **Huobao 短剧引擎**的集成逻辑，保证生成的稳定性和容错性。

## 2. Happy Path (黄金流程)
- [ ] **用例 1: 提取分镜** - 在 `/studio` 输入包含 3 个角色的 500 字小说段落，点击"提取"，确保页面在 30 秒内正确渲染至少 3 张分镜卡片，且字段填充完整 (`prompt` 和 `dialogue`)。
- [ ] **用例 2: 编辑分镜** - 修改第 2 张分镜卡片的对白文字，刷新页面或切换 Tab 后数据状态不丢失（暂存于客户端或草稿机制）。
- [ ] **用例 3: 触发视频** - 点击 "一键生成短剧"，进度条出现并在流程结束时提供一个真实可播放的 `.mp4` 链接。

## 3. Unhappy / Edge Cases (边界极限测试)
- [ ] **用例 4: Token 耗尽/拒绝** - 修改配置填入假的/失效的 API Key，点击"提取"，前端应在 5 秒内弹出极其显眼的红色报错 Toast，提示："模型服务拒绝访问，请检查 Key"。
- [ ] **用例 5: 未填内容空跑** - 在尚未输入小说正文时，点击提取，按钮应当置灰 (Disabled)，或者弹出微交互提示。
- [ ] **用例 6: 回调超时 (Timeout)** - 模拟 Huobao 后端阻塞（响应超过 60s），前端页面不会卡死，能够中断请求并显示 "后端无响应，请重试"。
- [ ] **用例 7: 断网场景** - 录入一段分镜时断开网络，确保草稿自动离线缓存在 localStorage，重连后可再次发起。
